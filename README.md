# StreamLine

StreamLine is an AI sensemaking dashboard designed to help users understand complex public issues beyond a personalized news feed.

Instead of optimizing only for relevance or engagement, StreamLine organizes information around decision-making. For a given topic, it shows what happened, what is contested, what different perspectives believe, how confident the system is, and what outcomes users should watch next.

Core Features

Event Brief: concise summary of what happened and why it matters
Structured Disagreement: consensus, contested views, dissenting views, and unknowns
Confidence Layer: simple confidence labels based on source agreement and uncertainty
Prediction Layer: rough probabilities for possible outcomes
Follow-up Questions: users can interrogate the brief interactively
Watchlist: key signals to monitor over time
Course Context

This project was created for BUSGEN 116: Governing the Digital World. It explores how AI agents can support democratic sensemaking by representing uncertainty, disagreement, and credibility instead of simply personalizing information feeds.

Long-Term Vision

StreamLine could expand from a consumer news dashboard into a decision intelligence platform for teams tracking policy, markets, technology, regulation, and geopolitical risk.
